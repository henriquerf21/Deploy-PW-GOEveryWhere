# PWA

Aplicação PWA do projeto (estafetas / entregas).

## O que está aqui

- **`estafeta-app/`** — única app Vue + Vite nesta pasta. Estrutura típica:
  - `src/views/` — ecrãs ligados às rotas
  - `src/components/` — componentes reutilizáveis
  - `src/stores/` — estado (Pinia ou equivalente)
  - `src/router/` — definição de rotas

## Como correr

```bash
cd estafeta-app
npm install
npm run dev
```

O servidor de desenvolvimento usa a porta **5175** (ver `package.json`).
