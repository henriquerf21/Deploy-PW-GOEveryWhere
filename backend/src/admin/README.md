# Strapi Admin (UI)

O painel em `/admin` é compilado pelo Strapi (React).

- `tsconfig.json` nesta pasta é **só para o build do admin**, não para a API em `src/api/`.
- Exemplos de extensão: `app.example.jsx`, `vite.config.example.js`.

A API do projeto corre em **JavaScript** — ver `backend/jsconfig.json` e `backend/README.md`.
